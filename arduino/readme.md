## Conexiones

| Arduino       | nRF          |
| :------------:|:------------:|
| D3            | MOSI         |
| A1            | CSN          |
| 3V3           | VCC          |
| A0            | MISO         |
| D4            | SCK          |
| D5            | CE           |
| GND           | GND          |
| No conectar   | IRQ          |
